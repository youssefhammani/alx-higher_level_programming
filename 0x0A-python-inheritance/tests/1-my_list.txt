this module define method that print list of integer sorted

Usage
=====

'''print_sorted()''' prints the list in ascending sort

>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

>>> print(my_list)
[1, 4, 2, 3, 5]

>>> my_list = MyList("hello")

>>> my_list
['h', 'e', 'l', 'l', 'o']

>>> my_list.print_sorted()
['e', 'h', 'l', 'l', 'o']

>>> my_list = MyList()

>>> my_list.print_sorted()
[]
